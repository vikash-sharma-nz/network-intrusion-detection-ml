# Dataset

## CICIDS-2017 Intrusion Detection Dataset

This project uses the CICIDS-2017 dataset developed by the Canadian Institute for Cybersecurity (CIC).

The dataset contains realistic network traffic data including:

- Normal network activity
- Denial of Service (DoS) attacks
- Distributed Denial of Service (DDoS) attacks
- Brute Force attacks
- Web attacks
- Infiltration attacks
- Botnet activity

---

## Dataset Size

The complete dataset is approximately 700 MB and is not stored in this repository due to GitHub storage limitations.

---

## Download Dataset

---

## Data Processing

The dataset processing workflow includes:

1. Data collection
2. Data cleaning
3. Missing value handling
4. Feature selection
5. Class balancing using SMOTE
6. Model training
7. Performance evaluation
